## Client work

These projects were developed for third-party companies, so the codebases are private. I can discuss the architecture, technical decisions, and business workflows without exposing confidential implementation details.

### CopaKMX — Tournament Management Platform

Full-stack platform for a corporate football championship with a public website and a protected admin panel.

**Built with:** Node.js, TypeScript, Express, PostgreSQL, Angular, PrimeNG, Tailwind CSS, JWT, image processing, and Google Cloud Platform.

**What it includes:** championship and sub-championship modeling, teams, companies, players, matches, results, standings, media management, carousel images, banners, and public schedules.

### Frenopartes — Workshop Management System

Internal multi-user system for managing vehicle workshop operations.

**Built with:** React, Vite, MUI, Chart.js, Node.js, Express, MongoDB, Mongoose, JWT, PDF generation, image processing, and PM2.

**What it includes:** vehicle entries, customer data, services, role-based access, entry/exit certificates, signatures, photos, warranties, dashboards, metrics, and audit-oriented workflows.
